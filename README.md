# sour-candy-taste-data

Clone repository to your computer.

Make a folder in the repository called 'Raw_Data'. Put a .csv file into it called 'sour-candy-data.csv'.

Run python script and the data from your sheet will be put into this [folder](/Processed_Data).

Our candy is there as an example, but we did not upload our initial data due to the fact it had individuals private data.
