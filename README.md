# sour-candy-taste-data

Clone repository to your computer.

Make a folder in the repository called 'Raw_Data'. Put a .csv file into it called 'sour-candy-data.csv'.

Run different python scripts based on which chart you would like to make and the data from your sheet will be put into this [folder](/Processed_Data).

pychart_maker_%of_data.py = Makes a pie chart based on the percentages of sample size compared to each other with how many samples there are per candy.

Our candy is there as an example, but we did not upload our initial data due to the fact it had individuals private data.
